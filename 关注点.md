# 关注点

---

> 由于课程的重点是架构设计,在征求过老师的同意后,功能需求只考虑 **搜索**, **评论** 这几个需求.广告植入,支付等功能没有质量约束,不进行设计.

| ID | 关注点 | 类型 | 描述 | 灵活性 |
| :--- | :--- | :--- | :--- | :--- |
| F1 | 搜索 | 功能需求 | 系统能通过关键字搜索各大电商网站的商品 | 小,系统的核心需求 |
| F2 | 评论 | 功能需求 | 用户能对商品发表评论 | 可以简化,比如只允许文字评论,不能贴图或打分等. |
| F3 | 可靠性 | 质量需求 | 数据存储出现故障要能继续工作并快速恢复 |  |
