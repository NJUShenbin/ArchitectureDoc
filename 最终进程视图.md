## 进程间接口描述

#### Browser与Firewall
* 通过HTTP协议通信

Firewall供接口
```
GET|POST /**
```

#### Firewall与Gateway
* 通过HTTP协议通信

Gateway供接口
```
GET|POST /**
```

#### Gateway与SearchService
* 通过HTTP协议通信

SearchService供接口
```
GET /search
Params: key
```

#### Gateway与StaticContentServer
* 通过HTTP协议通信

StaticContentServer供接口
```
GET /**.*
```
#### Gateway与CommentSerice
* 通过HTTP协议通信

CommentServer供接口
```js
POST /comment
Params:
{
    userId,
    goodsId,
    content
}
```
